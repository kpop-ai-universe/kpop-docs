# How to use mvsep.com to have clean acapella and extract vocals by ShadowMikuu

Link: https://mvsep.com/en

## Step 1
Click on **MDX-B** for the separation type, and then click **UVR-MDX-NET-Voc_FT (SDR: 9.64)**. I like using MDX-B to filter acapellas. The output encoding doesn't matter, but I mainly use flac.

## Step 2
Click on the **Separate** button and wait a while. After the instrumental and acapella have been separated, download the file that says **"Vocals"**.

## Step 3
Go back to the main page of MVSEP and set the separation type to **Ultimate Vocal Remover VR**. Then for the model type, click on **UVR-De-Echo-Normal**. Put the aggressiveness to **1.0** for the best quality. Then click on the **Separate** button and wait a while longer.

![Here](https://files.catbox.moe/tupv5v.png)

## Step 4
After separating the vocals, download the file that says **"Instruments"**, as the file that says **"Vocals"** has the reverbed vocals.

## Step 5
Go back to the main page of MVSEP and change the model type to **UVR-DeEcho-DeReverb**. Repeat the process from Steps 3 and 4. Afterward, you should have a clean acapella.

## Step 6
Return to the main page of MVSEP and change the separation type to **MDX-B Karaoke (lead/back vocals)**. Set the karaoke model type to **"Extract from vocals part (SDR lead vocals: 7.94)"**. It doesn't matter which output encoding you use. Click on the **Separate** button and wait for your vocals to be extracted.

## Step 7
After the vocals have been extracted, you will see five files. Download only the lead and backing vocals.

## Step 8
Return to the main page of MVSEP and change the separation type to **Medley Vox** if Dahyun isn't working for you. Then set the model type to **"Extract vocals first"**. Import the files one at a time, and start with either the lead or backing vocals. Again, the output encoding doesn't matter. Then click on separate and wait a little longer.

## Step 9
Download both files after they've been separated. Do the same process for the other vocals, depending on which vocals you separated first.
There's no limit duration.

## Step 10
Have fun creating AI covers and AI voice models, even if Dahyun is down!
